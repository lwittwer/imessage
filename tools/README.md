# Hardware Key Extraction Tools to Generate Validation Data

The prebuilt `ExtractKey.app.zip` (GUI) and `extract-key-cli.zip` (CLI) binaries are
attached to the [`tools-v1` release](https://github.com/lwittwer/imessage/releases/tag/tools-v1)
instead of being committed here — they're built artifacts, not source, and committing
a fresh zip on every rebuild bloated the repository permanently.

Download them from:

- GUI app: https://github.com/lwittwer/imessage/releases/download/tools-v1/ExtractKey.app.zip
- CLI: https://github.com/lwittwer/imessage/releases/download/tools-v1/extract-key-cli.zip

See the root [README](../README.md#step-1-extract-hardware-key-one-time-on-a-mac) for
usage instructions.
